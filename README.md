# MyHome

Sistema de anúncios de imóveis em Java. Permite cadastrar usuários, criar anúncios (Casa, Apartamento, Terreno) para **venda**, **aluguel** ou **temporada**, com valores para cada tipo de oferta; detalhes do imóvel (metros quadrados, descrição e suítes); buscar, visualizar e favoritar anúncios; conversar por chat sobre um anúncio; e realizar compra. Inclui formas de pagamento (Strategy: cartão, PIX, boleto) e planos de assinatura (Básico, Premium, Ilimitado). Há moderação de anúncios (estados Rascunho → Moderação → Ativo), notificação por e-mail/SMS (Observer + Adapter), log de mudanças de estado e configuração centralizada em `config.properties`. Dados iniciais podem ser carregados de CSV (`dados/usuarios.csv`, `dados/anuncios.csv`).
